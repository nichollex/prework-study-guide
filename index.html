Prework Study Guide

HTML

  • HTML adds structure to the content on a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Cascading style sheets are used to style a webpage.
  • Inline CSS, internal CSS style sheet, external CSS style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside of an element.

Git

  • Command ls: list contents of current working directory.
  • Command cd: change directories.
  • Git status to check what branch we are currently on.
  • Git checkout -b branch-name to create a new branch name and switch to it.
  • Git commit -A to commit all files in working branch.
  • Git commit -m "message here" to associate a message with a commit.
  • Important to frequently sync git bash and github.
